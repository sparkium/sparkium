This is the starting point for your journey with Sparkium. Here you will find the first steps to get started with Sparkium and create your eLearning materials.

## What can you expect?

We try to break down the learning process into small, manageable steps, so even if you are new to this kind of tool, you can follow along without feeling overwhelmed. Our goal is to explain the concepts with examples and practical advice, so you can quickly get up to speed and start creating your own materials.

To make it easier to follow all explanations contain examples that follow the consistent theme of a course about "Making a safe bonfire in Switzerland". True to the name, we will end up with something sparkling. So let's get started!

## What do I need to set up?

Actually you don't need much to get started. Just create a new folder on your computer and start adding subfolders and files to it. Of course there are more comfortable ways to do so. You can use your favorite code editor, like [Visual Studio Code](https://code.visualstudio.com), [Zed](https://zed.dev), [Sublime Text](https://www.sublimetext.com), or even a simple text editor. It becomes handy if you use an editor that supports Markdown, JSON and YAML, as these are the formats we will be working with. Another great feature to look for in an editor is the ability to manage multiple files and folders from within the editor itself, so you can easily manage your materials without needing to switch back and forth between different applications.
