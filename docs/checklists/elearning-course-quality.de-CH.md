# eLearning Kursqualitätscheckliste

- Einheitliche Terminologie verwenden:
  - `Lernende` für Personen, die an einem Kurs teilnehmen.
  - `Aufgabe` für einzelne Aktivitäten oder Schritte die von Lernenden bearbeitet werden muss.
  - `Vorschlag` für Lösungen oder Vorgehensweisen, die Lernende zur Bearbeitung einer Aufgabe verwenden können.
  - `Quiz` für Fragen, die Lernende beantworten müssen, um ihr Verständnis zu überprüfen.
- Bestandteile referenzierbar machen:
  - Jede Aufgabe, jeder Vorschlag und jedes Quiz sollte eine eindeutige ID haben, damit sie leicht referenziert werden können.
  - Teilschritte, Fragen und andere Einzelteile sollten ebenfalls referenzierbar sein, um eine klare Struktur zu gewährleisten.
